# HintBox Component Examples

This file demonstrates the different HintBox variants available for use in Better Ecology wiki pages.

## Did You Know

<HintBox type="didYouKnow">
  Cows will look around at their neighbors before deciding to move! This is called a quorum decision.
</HintBox>

## Hint

<HintBox type="hint">
  Watch closely when animals are grazing - they use a "giving up density" to decide when to move to a new patch of grass.
</HintBox>

## Try This

<HintBox type="tryThis">
  Watch a mother cow with her calf - notice how the calf mirrors the mother's movements!
</HintBox>

## Tip

<HintBox type="tip">
  Use the /debug eco command to visualize behavior states and see what your animals are thinking!
</HintBox>

## Fun Fact

<HintBox type="funFact">
  Chickens use the same flocking algorithms that real birds use in nature - based on boids research from 1986!
</HintBox>

## Multiple Hints

You can use multiple hints in a row:

<HintBox type="didYouKnow">
  Wolves hunt in packs using coordinated strategies, just like real wolves!
</HintBox>

<HintBox type="tryThis">
  Try watching a pack of wolves hunt - you'll see them split up and surround their prey.
</HintBox>

<HintBox type="funFact">
  The predation behavior implements research-based cooperative hunting strategies!
</HintBox>
