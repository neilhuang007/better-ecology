## Overview

| Attribute | Value |
|-----------|-------|
| Health | 20 HP (10 hearts) |
| Movement | Flying |
| Social Type | Attracted to players |
| Special Ability | Item collection |

## Behaviors

### Item Collection

Allays collect and deliver items.

- Hold and seek specific items
- Deliver to players or note blocks
- Search nearby area for matching items
- Stack collection over time

### Music Attraction

Allays are attracted to note blocks.

- Fly toward playing note blocks
- Deliver items to note block location
- Remember note block positions
- Dance near music

### Duplication

Allays can duplicate when given amethyst.

- Requires amethyst shard
- Must hear jukebox music
- Creates new allay
- Cooldown after duplication

### Following

Allays follow players who give them items.

- Stay within range of player
- Return to deliver collected items
- Can be redirected to note blocks

## Scientific Basis

Allay behavior is fantasy-based but inspired by symbiotic relationships and helper animal concepts.
