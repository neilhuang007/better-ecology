## Behaviors

### Hunting

Ocelots stalk prey using jungle cover.

| Parameter | Value |
|-----------|-------|
| Stalk speed | 0.3 |
| Detection range | 16.0 blocks |
| Pounce distance | 4.0 blocks |
| Hunt success rate | Higher in jungle biomes |

### Stealth

Ocelots move quietly through vegetation.

- Reduced detection by prey
- Uses cover for stalking approach
- Patient hunting style

### Flee Response

Ocelots are skittish around players.

- Flee distance: 6-8 blocks from player
- Reduced flee distance when player is crouching
- Trust builds over time with fish feeding

## Scientific Basis

Based on ocelot hunting behavior in tropical forests, including their preference for cover and patient stalking approach.
