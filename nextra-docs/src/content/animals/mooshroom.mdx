## Overview

| Attribute | Value |
|-----------|-------|
| Health | 10 HP (5 hearts) |
| Habitat | Mushroom fields |
| Variants | Red, Brown |
| Special | Mushroom stew production |

## Behaviors

### Stew Production

Mooshrooms can be milked for stew.

- Milk with bowl for mushroom stew
- Brown mooshrooms produce suspicious stew
- Flower feeding affects suspicious stew type

### Flower Conversion (Brown)

Brown mooshrooms can be fed flowers.

- Eat any small flower
- Next suspicious stew has flower effect
- One flower per stew

### Lightning Conversion

Mooshrooms change variant when struck by lightning.

- Red becomes Brown
- Brown becomes Red

### Shared Cow Behaviors

Mooshrooms share all cow behaviors:

- Herding
- Grazing patterns
- Maternal behaviors
- Quorum movement

See [Cow](/animals/cow) for full behavior details.

Mooshroom behavior extends cow behavior with fantasy mushroom symbiosis elements.
